# Architecture Design Docs

This directory contains canonical architecture design documents for QLanalyser Online.

Use `docs/templates/architecture_design_doc.md` for new architecture topics.

Rules:

- Repository Markdown docs are the source of truth.
- Feishu summaries are generated from these docs.
- Architecture decisions that affect multiple modules must also be recorded in `docs/DECISIONS.md`.


## Current canonical architecture docs

- `docs/architecture/system_architecture.md`: system architecture source of truth.
- `docs/architecture/version_detailed_design.md`: version-by-version detailed design and roadmap boundaries.
