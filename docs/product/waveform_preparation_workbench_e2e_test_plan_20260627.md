# 数据准备交互式波形工作台端到端测试文档

版本：2026-06-27

## 1. 测试目标

证明数据准备页不是静态预览，而是可操作的波形工作台。测试必须覆盖教学模式内置数据、波形自动加载、浏览、滤波预览、参考设置、坏道、坏段、恢复和后续分析门禁。

## 2. 测试环境

- 前端：`http://127.0.0.1:4174/index.html?customer_demo=auto&api=http%3A%2F%2F127.0.0.1%3A8001%2Fapi`
- 后端：`http://127.0.0.1:8001/api`
- 浏览器：Playwright / Edge 或 Chromium
- 数据：教学模式内置 EEG 数据；必要时使用合成 FIF/EDF 数据。

## 3. 测试用例

### T-WF-01 教学数据自动波形预览

步骤：
1. 打开客户工作台。
2. 登录或自动登录。
3. 进入教学模式。
4. 结束蒙版引导。
5. 查看数据准备页 canvas。

期望：
- 教学 banner 存在。
- `#eegCanvas` 非空。
- 没有触发上传请求。
- 预览请求使用教学文件。

### T-WF-02 波形浏览控件

步骤：
1. 点击下一段。
2. 点击上一段。
3. 点击放大、缩小、复位。
4. 调整起点、窗口、增益、通道数。

期望：
- canvas 更新。
- 摘要区显示当前窗口。
- 控件值与内部状态一致。

### T-WF-03 鼠标拖拽选段

步骤：
1. 在 canvas 上拖拽。
2. 查看起止时间输入框。

期望：
- 选段高亮显示。
- `#segmentStart` 和 `#segmentEnd` 更新。
- 摘要区提示已选择片段。

### T-WF-04 剔除片段与恢复

步骤：
1. 选中片段。
2. 点击剔除片段。
3. 点击恢复片段。

期望：
- 剔除后出现红色坏段叠加。
- 恢复后记录恢复动作。
- 原始 EEG 不被改写。

### T-WF-05 坏道标记与恢复

步骤：
1. 点击标记坏道。
2. 点击恢复坏道。

期望：
- 摘要区显示坏道修改数量。
- 恢复动作被记录。

### T-WF-06 滤波预览

步骤：
1. 打开滤波预览。
2. 修改低切/高切/陷波。
3. 查看预览 note 和 canvas。

期望：
- 滤波预览状态显示为开启。
- 参数变化触发预览更新或状态更新。
- 文案说明不改写原始 EEG。

### T-WF-07 重参考设置

步骤：
1. 切换平均参考、指定通道参考、双极参考。
2. 输入参考通道和双极通道对。
3. 确认数据准备。

期望：
- 重参考设置在数据准备方案中可追踪。
- 页面不把参考方案显示为分析方法。

### T-WF-08 教学沙盒分析联动

步骤：
1. 完成教学沙盒引导。
2. 直接进入分析任务。
3. 点击 PSD。

期望：
- `/api/tasks` 使用 `proj_demo_learning` 和 `eeg_demo_teaching_oddball`。
- 不触发上传。
- 教学模式仍保持。

## 6. 发布门禁

以下任一失败不得通过：

- 教学数据无波形。
- 波形无法拖拽选段。
- 剔除/恢复无记录。
- 滤波/参考控件不在数据准备页。
- PSD 教学沙盒分析触发上传。
- 页面出现正向医疗诊断、治疗或临床决策文案。
