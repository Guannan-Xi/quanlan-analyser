# 数据准备交互式波形工作台详细设计

版本：2026-06-27

## 1. 设计原则

1. 波形是主界面，不是附属预览。
2. 预处理决策必须在波形旁边完成。
3. 原始数据只读，所有修改进入数据准备记录。
4. 用户文案面向科研人员，不出现 fixture、workflow、backend 等开发词。
5. QC 与重参考属于数据准备，不进入分析方法列表。

## 2. 页面结构设计

```text
数据准备
├─ 当前项目/数据状态
├─ 交互式波形工作台
│  ├─ 顶部工具条
│  │  ├─ 上一段 / 下一段 / 重新加载
│  │  ├─ 放大 / 缩小 / 复位
│  │  ├─ 起点 / 时间窗 / 增益 / 通道数
│  │  └─ 滤波预览 / 片段 / 坏道 / 参考快捷入口
│  ├─ 波形画布
│  ├─ 波形状态摘要
│  ├─ 事件/提示条
│  ├─ 操作摘要条
│  ├─ 片段、标签、坏道操作卡
│  └─ 右侧预处理设置
│     ├─ 重参考设置
│     ├─ 参考通道
│     ├─ 双极通道对
│     ├─ 低切 / 高切 / 陷波
│     └─ 保存/恢复/确认
└─ 数据准备说明与下载
```

### 3.2 窄屏

波形在上，预处理设置紧跟在波形后，不能被放到页面很深的位置。工具条允许换行。

## 4. 交互设计

### 4.1 自动加载

- 选择 EEG 文件后调用现有快速预览任务。
- 教学模式进入后自动选择内置文件并自动预览。
- 加载中显示骨架波形，避免空白误解。

### 4.2 浏览

- 上一段/下一段：按当前窗口长度的 80% 平移。
- 放大/缩小：改变窗口长度。
- 复位：回到 0 秒，10 秒窗口，2x 增益，8 通道。
- 滚轮：平移。
- Ctrl/Cmd + 滚轮：缩放。

### 4.3 片段操作

- 在 canvas 上拖拽生成选区。
- 剔除片段写入草稿记录并以红色半透明区显示。
- 恢复片段写入恢复记录。

### 4.4 坏道操作

- 标记坏道写入草稿记录。
- 恢复坏道写入恢复记录。
- 摘要区显示当前坏道草稿数量与最近动作。

### 4.5 滤波和参考

- 滤波预览开关打开后重新生成当前窗口预览。
- 低切、高切、陷波参数变化时，如滤波预览开启则刷新波形。
- 重参考设置保存在数据准备方案参数中。
- 参考通道和双极通道对不做源定位承诺。

## 7. 视觉规范

- 波形画布保持白底或浅底，时间轴和通道名清楚。
- 坏段使用红色半透明叠加，不遮挡波形。
- 当前选段使用粉色/紫色半透明叠加，与坏段区分。
- 控件使用项目既有按钮、卡片、圆角和阴影 token。
- 科学图不使用彩虹/jet 作为默认定量配色。

## 8. 风险与缓解

- 风险：大文件预览过慢。缓解：只加载当前窗口，必要时使用降采样预览。
- 风险：用户误以为滤波已改写原始数据。缓解：面板持续显示“只预览，不改写原始 EEG”。
- 风险：事件相关分析在无事件数据上误运行。缓解：分析任务门禁检查事件/epoch 记录。
